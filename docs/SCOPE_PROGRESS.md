## AdSense審査落ち → 品質改善（2026-05-31 実施）

不承認理由: **「有用性の低いコンテンツ」**（管理画面の[サイト]ページに明記）。
原因: 無料Llamaでの量産記事＝薄い・事実誤認・無関係アフィリンク・全テンプレ同型。

対応内容（本番デプロイ済み）:
- 低品質記事を **25本削除**（47→22本）＋後発の自動投稿不良1本を差替 → **計23本**
- 主要 **19本をリライト**（事実検証・独自性・2,500字超、料金は「公式で要確認」）
- 内部リンクの**全404を修復**（日付つきURL→slug形式）
- 無関係アフィリンク（旅行/免許/撮影/転職）を除去
- **自動投稿を一時停止**（daily/evening の cron 無効化）
- 生成スクリプトの内部リンクURLバグ修正＋最低文字数2500字に
- About に編集方針・免責・運営者情報を追記（E-E-A-T）
- 記事生成せず公開する `deploy.yml` を追加（main push で自動デプロイ）

追加の品質仕上げ（同日）:
- ai-転職: 重複表・水増し・削除記事への参照を排除し全面リライト（→2,528字）
- ai副業-始め方: 反復語「意外な事実」を是正、Cursor URL更新
- claude-chatgpt比較: 陳腐化しやすいモデル名を撤去し「公式で確認」表現に
- → テンプレ語「意外な事実」はサイト全体から消去済み

Search Console準備状況: sitemap.xml（26 URL・全記事含む）・robots.txt（全許可＋sitemap参照）とも**健全。修正不要**。登録はユーザー操作のみ。

ライブ検証: 削除記事→404 / リライト記事→200 / 新内容反映 OK。

- 記事生成: Groq API - Llama 3.3 70B versatile（無料）

## 運用上の重要注意

- ⚠️ **自動投稿は止めたまま**にする（再開すると低品質記事で再び落ちる）。記事追加は手動・高品質で。
